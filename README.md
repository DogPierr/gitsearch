# Git search

Консольное приложение, которое подссчитывает статистики по данному git репозиторию.

```
Подсчёта статистик авторов git репозитория. Все статистики считаются для состояния репозитория на момент конкретного коммита.

Usage:
  gitsearch [flags]

Flags:
  -d, --exclude string       набор Glob паттернов, исключающих файлы из расчёта, например 'foo/*,bar/*'
  -e, --extensions string    список расширений, сужающий список файлов в расчёте; множество ограничений разделяется запятыми, например, '.go,.md'
  -f, --format string        формат вывода; один из tabular (дефолт), csv, json, json-lines (default "tabular")
  -h, --help                 help for gitsearch
  -l, --languages string     список языков (программирования, разметки и др.), сужающий список файлов в расчёте; множество ограничений разделяется запятыми, например 'go,markdown'
  -o, --order-by string      ключ сортировки результатов; один из lines (дефолт), commits, files (default "lines")
  -r, --repository string    путь до Git репозитория; по умолчанию текущая директория (default ".")
  -x, --restrict-to string   набор Glob паттернов, исключающий все файлы, не удовлетворяющие ни одному из паттернов набора
  -v, --revision string      указатель на коммит; HEAD по умолчанию (default "HEAD")
  -c, --use-committer        булев флаг, заменяющий в расчётах автора (дефолт) на коммиттера
```
